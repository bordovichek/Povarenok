◆ create_access_token()

str security.create_access_token ( int user_id)

Создаёт JWT-токен доступа для аутентифицированного пользователя.

Токен используется для авторизации запросов к API. Содержит ID пользователя и срок действия (по умолчанию 7 дней).

Parameters
user_idID пользователя в базе данных.
Returns

Создаёт одноразовый токен для сброса пароля.

Токен содержит уникальный JTI для предотвращения повторного использования и имеет короткий срок действия (по умолчанию 20 минут).

◆ decode_token()

TokenPayload security.decode_token ( str token)

Расшифровывает и валидирует JWT-токен.

Проверяет подпись токена и срок действия. В случае невалидного токена выбрасывает исключение jwt.PyJWTError.

Parameters
tokenJWT-токен в виде строки.
Returns
TokenPayload Объект с данными из токена.
Exceptions
